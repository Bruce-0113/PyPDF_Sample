# PyPDF_Sample
 This repository store some example codes of PyPDF2 packages.
 - PDF_Merge.py : Merge PDF files.
 - PDF_rotate.py : Rotate PDF files.
 - PDF_watermarker.py : Add watermark on template PDF file.
